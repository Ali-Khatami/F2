APP

String final static

Defined in sdk/src/constants.js:17

The APP class should be applied to the DOM Element that surrounds the entire App, including any extra html that surrounds the APP_CONTAINER that is inserted by the Container. See appWrapper property in the F2.ContainerConfiguration object.

APP_CONTAINER

String final static

Defined in sdk/src/constants.js:28

The APP_CONTAINER class should be applied to the outermost DOM Element of the App.

APP_TITLE

String final static

Defined in sdk/src/constants.js:37

The APP_TITLE class should be applied to the DOM Element that contains the title for an App. If this class is not present, then setTitle will not function.

APP_VIEW

String final static

Defined in sdk/src/constants.js:47

The APP_VIEW class should be applied to the DOM Element that contains a view for an App. The DOM Element should also have a F2.Constants.Views.DATA_ATTRIBUTE attribute that specifies which F2.Constants.Views it is.

APP_VIEW_TRIGGER

String final static

Defined in sdk/src/constants.js:59

APP_VIEW_TRIGGER class should be applied to the DOM Elements that trigger an F2.Constants.Events.APPVIEWCHANGE event. The DOM Element should also have a F2.Constants.Views.DATA_ATTRIBUTE attribute that specifies which F2.Constants.Views it will trigger.

MASK

String final static

Defined in sdk/src/constants.js:73

The MASK class is applied to the overlay element that is created when the F2.UI\showMask method is fired.

MASK_CONTAINER

String final static

Defined in sdk/src/constants.js:82

The MASK_CONTAINER class is applied to the Element that is passed into the F2.UI\showMask method.
