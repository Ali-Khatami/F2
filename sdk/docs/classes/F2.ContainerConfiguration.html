appWrapper

Function

Defined in sdk\src\classes.js:222

Allows the Container to wrap an App in extra html. The function should accept an F2.App object and also a string of html. The extra html can provide links to edit app settings and remove an app from the Container. See F2.Constants.Css for CSS classes that should be applied to elements.

appWriter

Function

Defined in sdk\src\classes.js:233

supportedViews

Array

Defined in sdk\src\classes.js:256

Specifies what views a Container will provide buttons or liks to. Generally, the views will be switched via buttons or links in the App's header. The F2.Constants.Views.HOME view should always be present.
