registerApps

(
  • apps
  • [appAssets]
)

Defined in src\container.js:533

Begins the loading process for all Apps. The App will be passed the F2.App object which will contain the App's unique instanceId within the Container. Optionally, the F2.AppAssets can be passed in and those assets will be used instead of making a request.

Parameters:

  • apps Array

    An array of F2.App objects

  • [appAssets] Array optional

    An array of F2.AppAssets objects. This array must be the same length as the apps array that is objects. This array must be the same length as the apps array that is passed in. This can be useful if Apps are loaded on the server-side and passed down to the client.
