F2.Constants.Views Class

Defined in: sdk/src/constants.js:225
Module: f2

The available view types to Apps. The view should be specified by applying the F2.Constants.Css.APPVIEW class to the containing DOM Element. A DATAATTRIBUTE attribute should be added to the Element as well which defines what view type is represented. The hide class can be applied to views that should be hidden by default.
